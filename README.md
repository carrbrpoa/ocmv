# ocmv
Configureable Map Viewer using OpenLayers 3
A map viewer based on [CMV](https://github.com/cmv/cmv-app).

##Supported Type:
* WMS
 
##Current Widgets:
* Bookmarks
* Draw
* Growler
* Help
* Home button
* Identify Popup
* Measure
* Locate button
* Map info
* Overview map
* StreetView

##Client technology used:
* [Openlayers 3](http://openlayers.org/)
* [Dojotoolkit](http://dojotoolkit.org/)
